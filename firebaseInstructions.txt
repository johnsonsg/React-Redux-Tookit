## Firebase Setup Realtime Database

## Create Project
* Create New Project
* Give it a Name
* Click Continue
* You can disable Google Analytics to start
* Click on Create Project
* Then continue to Console

## Build Database
* In Build, Click on Realtime Database: Great for API's

* [ For Cloud Firestore, visit tutorial: https://firebase.google.com/products/firestore?gclid=Cj0KCQiA54KfBhCKARIsAJzSrdrAJLcQ9FMG17aCNiikE0mUI5FiHgukHrTXRomb4pCXhcJxJK-AnsUaAkvLEALw_wcB&gclsrc=aw.ds ]

* Create Database and Continue
* Select 'Start in test mode' and click enable
